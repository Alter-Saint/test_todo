#  To-Do List

Це реалізація тестового завдання на позицію Junior Frontend Developer. Додаток дозволяє керувати списком завдань з локальним збереженням даних та можливістю редагування.

#  Технологічний стек

* **Framework:** Next.js 14 (App Router) 

* **Language:** TypeScript 

* **Styling:** Tailwind CSS 

* **State Management:** `useReducer` (Бонусний пункт) 

* **Storage:** LocalStorage 


#  Реалізований функціонал

* **Керування завданнями:** Додавання, видалення та відмітка про виконання.


* **Редагування:** Можливість змінити текст завдання за допомогою **подвійного кліку** (Бонусний пункт).


* **Збереження даних:** Всі зміни автоматично зберігаються в `localStorage` та відновлюються при перезавантаженні сторінки.


* **Лічильник:** Динамічне відображення кількості невиконаних завдань.


* **Індикація завантаження:** Мінімалістичний екран завантаження під час ініціалізації стану на клієнті.


* **Адаптивність:** Інтерфейс коректно відображається на мобільних пристроях та десктопах.



# Команди для запуску

1. **Встановлення залежностей:**
```bash
npm install

```


2. **Запуск у режимі розробки:**
```bash
npm run dev

```


3. **Збірка проекту:**
```bash
npm run build

```



 Можливі покращення (якби було більше часу) 

1. **Фільтрація:** Додавання фільтрів (Всі / Активні / Виконані).
2. **Анімації:** Використання *Framer Motion* для плавного додавання та видалення елементів.
3. **Валідація:** Інтеграція *React Hook Form* та *Zod* для більш гнучкої перевірки вводу.
4. **Backend:** Перехід з `localStorage` на повноцінний API з базою даних для синхронізації між пристроями.
